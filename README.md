#### Question 1
What is Data?
Data is defined as raw, unprocessed facts, figures, symbols, or observations that can be collected, stored, and analyzed to generate meaningful information. *examples* Business Analytics: Analyzing sales figures and customer demographics to forecast trends. Computing: Storing binary code, text, or multimedia files (images, videos) for processing.
#### Question 2
Explain the difference between: Structured Data and Unstructured Data
The main difference here is how the informantion is organized, *for instance* in structured Generally resides in relational databases (RDBMS) like MySQL, PostgreSQL, and SQL Server, or in data warehouses. and in unstractured Typically kept in data lakes, file systems, or NoSQL databases in its native format.
#### Question 3
What is a  Schema? Explain using a simple example.
A schema is a structured framework, plan, or blueprint that organizes information *for instance* The formal design or "skeleton structure" of a database, outlining tables, columns, and relationships. 
#### Question 4
Explain the difference between: A list and A dictionary
a list is an ordered collection accessed by a numerical index, while a dictionary is a collection of key-value pairs accessed using unique keys.
